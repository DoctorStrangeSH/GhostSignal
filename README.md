# 👻 Ghost Signal

> Every case leaves a trace in the static.

**Ghost Signal** — интерактивный детективный терминал в стиле нуар. 
Раскрывайте нераскрытые дела, исследуйте локации, допрашивайте свидетелей 
и собирайте улики на доске расследования.

---

## 🎮 Геймплей

- 📂 **Дела** — загружайте нераскрытые дела из архива
- 🗺️ **Карта города** — перемещайтесь между локациями
- 🔍 **Поиск улик** — осматривайте сцены и находите предметы
- 👥 **NPC** — звоните и допрашивайте свидетелей
- 📋 **Доска расследования** — соединяйте улики
- 🔐 **Мини-игры** — расшифровывайте коды, ищите предметы
- 🌙 **День/ночь** — время влияет на доступность локаций и NPC
- 📝 **Заметки** — сохраняйте важные фрагменты документов
